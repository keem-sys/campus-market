# CampusMarket
> **Project Management 3 (PRM372S)**  
> Cape Peninsula University of Technology (CPUT)  
> Faculty of Informatics & Design  

CampusMarket is a secure, mobile-first web application designed to connect students, faculty, residents, and verified local vendors within a trusted university campus ecosystem. The platform facilitates the safe buying, selling, and trading of second-hand textbooks, dorm essentials, electronics, and academic services while promoting affordability and campus sustainability.

---

## 🛠️ Tech Stack

*   **Frontend:** React.js, Tailwind CSS (Mobile-first, responsive web design)
*   **Backend:** Node.js (Express.js)
*   **Database:** MySQL
